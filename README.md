# Programação Orientada a Objetos

Este repositório reúne implementações desenvolvidas durante a disciplina de **Programação Orientada a Objetos** da Universidade Federal Fluminense (UFF), com foco na aplicação prática dos principais conceitos do paradigma orientado a objetos.

## Descrição

Os códigos foram construídos ao longo da disciplina com o objetivo de exercitar e consolidar conceitos fundamentais como modelagem de classes, encapsulamento, herança e polimorfismo, além do uso de boas práticas de organização de código.

## Conteúdo

O repositório inclui exercícios e exemplos envolvendo:

* Criação e organização de classes
* Encapsulamento e uso de métodos de acesso
* Herança e reutilização de código
* Polimorfismo
* Manipulação de coleções (ArrayList, HashMap, etc.)
* Estruturação em camadas (DAO, serviços)
* Leitura e escrita de arquivos
* Noções básicas de concorrência (threads)

## Objetivo

Consolidar o entendimento dos princípios da Programação Orientada a Objetos por meio da implementação prática, contribuindo para o desenvolvimento de software mais organizado, reutilizável e escalável.

## Tecnologias utilizadas

* Java
* Bibliotecas padrão da linguagem

## Como executar

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/programacao-orientada-a-objetos.git
   ```

2. Acesse o diretório:

   ```bash
   cd programacao-orientada-a-objetos
   ```

3. Compile e execute os arquivos desejados utilizando um compilador Java:

   ```bash
   javac NomeDoArquivo.java
   java NomeDoArquivo
   ```

## Organização

Os arquivos estão organizados de acordo com os tópicos abordados ao longo da disciplina, incluindo versões completas e incompletas de exercícios, permitindo acompanhar a evolução do aprendizado.
